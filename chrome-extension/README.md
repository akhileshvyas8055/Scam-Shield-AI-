# Chrome Extension - AI Scam Detector

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right corner)
3. Click "Load unpacked"
4. Select the `chrome-extension` folder
5. The extension icon will appear in your toolbar

## Usage

1. Click the extension icon in your Chrome toolbar
2. Paste the internship offer text
3. Optionally add email and website
4. Click "Analyze Now"
5. View instant AI verdict

## Features

- Instant scam detection
- Clean, minimal interface
- Works offline (once backend is running)
- Quick access from any webpage
- Color-coded results (Green/Yellow/Red)

## Note

Make sure the backend server is running on `http://localhost:5000` before using the extension.
